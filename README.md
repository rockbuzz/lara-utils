# Lara Utils

## Requirements

PHP >=7.4

## Install

```bash
$ composer require rockbuzz/lara-utils
```
## Usage

```php
Rockbuzz\LaraUtils\Casts\Password::class;
Rockbuzz\LaraUtils\Casts\Schemaless::class;
Rockbuzz\LaraUtils\Data\Base::class;
Rockbuzz\LaraUtils\Traits\HasActive::class;
```
